# 课表模块接入说明

## 功能概述

课表模块是一个纯前端实现的完整课程管理功能，支持周视图展示、课程增删改查、冲突检测、临时调课、CSV导入导出、倒计时提醒等功能。

## 文件结构

```
markdown-notebook/
├── index.html              # 主页面（已集成课表入口和样式）
├── schedule.js             # 课表模块核心代码
├── SCHEDULE_CSV_TEMPLATE.md  # CSV导入模板说明
└── SCHEDULE_README.md      # 本文档
```

## 使用方式

### 访问课表

1. **桌面端**: 点击左侧边栏的"我的课表"按钮
2. **移动端**: 点击底部导航栏的"课表"标签

### 添加课程

1. 点击"添加课程"按钮
2. 填写课程信息：
   - 课程名称（必填）
   - 教师、教室（可选）
   - 星期、节次（必填）
   - 周次范围（必填）
   - 单双周规则
   - 课程颜色
3. 点击保存

### 编辑/删除课程

- 点击课表上的课程卡片
- 修改信息后保存，或点击删除

### 周次切换

- 使用左右箭头切换周次
- 点击"回到本周"快速返回当前周
- 当前周会显示"本周"标签

### 导入/导出

#### 导入 CSV
1. 点击"导入/导出"按钮
2. 选择"导入 CSV"
3. 选择符合格式的 CSV 文件

#### 导出 CSV
1. 点击"导入/导出"按钮
2. 选择"导出 CSV"
3. 文件将自动下载

#### 导出图片
1. 点击"导入/导出"按钮
2. 选择"导出图片"
3. 生成高清 PNG 图片并下载

## CSV 导入格式

### 表头
```csv
课程名称,教师,教室,星期,开始节次,结束节次,开始周,结束周,单双周,颜色
```

### 示例
```csv
课程名称,教师,教室,星期,开始节次,结束节次,开始周,结束周,单双周,颜色
高等数学,张教授,A101,1,1,2,1,16,0,#4f46e5
大学英语,李老师,B203,1,3,4,1,16,0,#10b981
```

### 字段说明
- **星期**: 1=周一，2=周二，...，7=周日
- **单双周**: 0=每周，1=单周，2=双周
- **颜色**: 十六进制颜色代码，如 #4f46e5

详细说明见 `SCHEDULE_CSV_TEMPLATE.md`

## 数据存储

- 课程数据存储在 LocalStorage 中
- Key: `courseSchedule`
- 包括：课程列表、临时调课记录、学期开始日期

## 配置常量

在 `schedule.js` 顶部可修改：

```javascript
const SCHEDULE_CONFIG = {
    MAX_PERIODS: 12,        // 每天最大节次
    SEMESTER_WEEKS: 20,     // 学期总周数
    PERIOD_MINUTES: 45,     // 每节课时长
    BREAK_MINUTES: 10,      // 课间休息
    COLORS: [...]           // 课程颜色选项
};
```

## 注意事项

1. **时间计算**: 默认第一节课 8:00 开始，每节 45 分钟，课间 10 分钟
2. **学期开始**: 自动判断为每年 9 月 1 日或 2 月 20 日
3. **冲突检测**: 添加/编辑课程时会自动检测时间冲突
4. **临时调课**: 可在课程上标记停课或调课（开发中）
