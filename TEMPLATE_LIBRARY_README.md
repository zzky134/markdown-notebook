# Markdown 模板库功能说明

## 功能概述

模板库功能为用户提供预置的学科常用模板，降低使用门槛，解决用户不知道怎么写笔记、从零开始写的使用阻力。

## 预置模板

### 1. 英语单词笔记模板 📝
- **分类**: 语言学习
- **用途**: 记录英语单词的完整信息
- **包含字段**:
  - 单词、音标、词性、中文释义
  - 例句（带中文翻译）
  - 词组搭配
  - 同义词/反义词
  - 易错点（拼写、用法）
  - 记忆技巧

### 2. 公式定理笔记模板 📐
- **分类**: 数学/理科
- **用途**: 记录数学公式和定理
- **包含字段**:
  - 定理/公式名称
  - 文字描述和数学表达式
  - 适用条件
  - 推导思路（核心思路+步骤）
  - 典型例题（题目+解答）
  - 易错提醒
  - 相关公式

### 3. 作业计划模板 📚
- **分类**: 学习管理
- **用途**: 规划每日/每周学习任务
- **包含字段**:
  - 任务概览（目标、截止时间）
  - 任务清单（按优先级分类）
  - 时间安排表
  - 重难点记录
  - 完成情况总结
  - 复盘总结（优点、改进点、效率评分）

## 使用方式

### 方式一：从侧边栏打开
1. 在左侧边栏点击"模板库"按钮（网格图标）
2. 在弹出的模板库中选择合适的模板
3. 预览模板内容
4. 点击"使用此模板"插入到当前笔记

### 方式二：新建笔记时选择
（后续可扩展）

## 模板格式规范

- 使用 `【】` 标记需要用户填充的内容
- 清晰的标题层级结构
- 预留空白位置，拿来就能直接填空使用
- 排版美观，支持 Markdown 所有特性

## 技术实现

### 文件结构
```
markdown-notebook/
├── index.html          # 主页面（已集成模板库UI和逻辑）
├── templates.js        # 模板数据和管理器
└── TEMPLATE_LIBRARY_README.md  # 本文档
```

### 代码集成点

1. **UI 入口** (`index.html`):
   - 侧边栏新增模板库按钮
   - 模板库模态框
   - 模板预览模态框

2. **样式** (`index.html` 的 `<style>` 中):
   - `.template-library-modal` - 模板库弹窗样式
   - `.template-card` - 模板卡片样式
   - `.template-preview-modal` - 预览弹窗样式
   - 移动端适配样式

3. **逻辑** (`index.html` 的 `<script>` 中):
   - `openTemplateLibrary()` - 打开模板库
   - `renderTemplateLibrary()` - 渲染模板列表
   - `previewTemplate()` - 预览模板
   - `insertTemplateContent()` - 插入模板内容

4. **模板数据** (`templates.js`):
   - `markdownTemplates` - 模板数据对象
   - `TemplateManager` - 模板管理器类

## 扩展新模板

如需添加新模板，编辑 `templates.js` 文件：

```javascript
const markdownTemplates = {
    // 现有模板...

    // 新模板
    newTemplate: {
        id: 'newTemplate',
        name: '新模板名称',
        icon: '🎯',
        category: '分类名称',
        description: '模板描述',
        content: `# 标题

## 字段1: 【填充内容】

## 字段2: 【填充内容】
`
    }
};
```

## 注意事项

1. 模板使用 `【】` 作为占位符标记，用户使用时需要替换为实际内容
2. 模板内容支持所有 Markdown 语法（标题、列表、表格、代码块等）
3. 插入模板时会自动保存当前笔记
4. 模板插入后会自动更新预览

## 浏览器兼容性

- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持移动端浏览器
- 需要 JavaScript 支持
