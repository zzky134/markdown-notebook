# 课表 CSV 导入模板

## 文件格式

CSV 文件需使用 UTF-8 编码，第一行为表头，每行一门课程。

## 表头字段

```csv
课程名称,教师,教室,星期,开始节次,结束节次,开始周,结束周,单双周,颜色
```

## 字段说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 课程名称 | 是 | 课程名称 | 高等数学 |
| 教师 | 否 | 授课教师 | 张教授 |
| 教室 | 否 | 上课地点 | A101 |
| 星期 | 是 | 1-7 表示周一到周日 | 1 |
| 开始节次 | 是 | 1-12 | 1 |
| 结束节次 | 是 | 1-12，需≥开始节次 | 2 |
| 开始周 | 是 | 1-20 | 1 |
| 结束周 | 是 | 1-20，需≥开始周 | 16 |
| 单双周 | 否 | 0=每周，1=单周，2=双周 | 0 |
| 颜色 | 否 | 颜色代码或留空自动分配 | #4f46e5 |

## 示例 CSV 文件

```csv
课程名称,教师,教室,星期,开始节次,结束节次,开始周,结束周,单双周,颜色
高等数学,张教授,A101,1,1,2,1,16,0,#4f46e5
大学英语,李老师,B203,1,3,4,1,16,0,#10b981
程序设计,王工程师,机房C305,2,1,3,1,16,0,#f59e0b
数据结构,赵教授,A102,3,1,2,1,16,0,#ef4444
体育,刘教练,体育馆,4,5,6,1,16,2,#8b5cf6
马克思主义原理,陈老师,D201,5,1,2,1,16,0,#06b6d4
```

## 导入说明

1. 星期：1=周一，2=周二，...，7=周日
2. 节次：1-12 节课，支持跨节次（如 1-2 节连上）
3. 单双周：0=每周都有，1=仅单周，2=仅双周
4. 颜色：可填写十六进制颜色代码（如 #4f46e5），留空则自动分配
5. 文件编码：必须使用 UTF-8，否则中文会乱码

## 导出功能

课表模块支持：
- 导出整周课表为高清 PNG 图片
- 导出当前周课表数据为 CSV 文件
